# CPI Forecasting

This project explores time-series forecasting methods for the Bank of Canada Commodity Price Index (BCPI).

## What is included
- An R script for building and comparing ARIMA-style forecasting models
- A report PDF summarizing the analysis and interpretations

## Skills demonstrated
- time-series analysis
- data visualization
- statistical modeling
- forecasting and model comparison

## Files
- ARMA_Model_RCode.R — main R analysis script
- ARMA Models on BCPI.pdf — written report and results
